## 📊 知识体系概览

```
操作系统原理
├── 硬件基础 ──── 系统运行机制（中断、MMU、系统调用）
├── Linux基础 ──── 目录结构、C编程、编译工具链
├── 文件管理 ──── 逻辑结构 → 目录 → 外存组织 → 空闲管理 → Ext2
├── 进程管理 ──── 进程概念 → PCB → 状态转换 → fork/exec → 信号
├── 线程管理 ──── 线程概念 → Pthreads → 信号量 → P/V → 经典同步
├── 进程通信 ──── 管道 → 消息队列 → 共享内存 → IPC信号量
├── 网络编程 ──── Socket → TCP → Web服务器 → 并发方案
├── 处理机调度 ── 三级调度 → FCFS/SJF/RR/优先级 → 多级反馈队列
├── 死锁 ──────── 四条件 → 预防 → 避免(银行家) → 检测与解除
├── 存储管理 ──── 分页 → 分段 → 段页式 → 虚拟存储器 → 置换算法
├── I/O系统 ──── 控制方式 → 中断/DMA → 缓冲 → 设备驱动
└── 代码优化 ──── CPE → 代码移动 → 强度削减 → 消除别名
```
