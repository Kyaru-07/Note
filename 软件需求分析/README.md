## 📝 笔记结构说明

每章笔记均采用 **7步结构**：

1. **概念引入** — 建立直觉，理解为什么需要这个知识
2. **核心概念/公式** — 关键定义、规则、关系
3. **方法速通** — 实用技巧和操作步骤
4. **典型示例** — 结合课件案例的详细解析
5. **真题/实战** — 实际应用场景和练习
6. **巩固练习** — 自测题目（含参考答案）
7. **总结复盘** — 知识要点、易错点、考试重点
